    Enum: UpdateFrequency

    The frequency at which a dataset is updated with new data.

    Based on Dublin Core Collection Description Frequency Vocabulary.

    __

    URI: valuesets:UpdateFrequency

    Permissible Values

    Value Title Meaning Description Iso Duration
    CONTINUOUS dcterms:Frequency Data is updated continuously or in real-time. PT0S
    DAILY Data is updated once per day. P1D
    TWICE_WEEKLY Twice Weekly Data is updated twice per week. P3D
    WEEKLY Data is updated once per week. P1W
    BIWEEKLY Data is updated every two weeks. P2W
    MONTHLY Data is updated once per month. P1M
    BIMONTHLY Data is updated every two months. P2M
    QUARTERLY Data is updated once per quarter (every three months). P3M
    SEMIANNUAL Data is updated twice per year. P6M
    ANNUAL Data is updated once per year. P1Y
    BIENNIAL Data is updated every two years. P2Y
    TRIENNIAL Data is updated every three years. P3Y
    IRREGULAR Data is updated at irregular intervals.
    NEVER Data is not updated after initial publication.
    Historical or archival datasets.
    UNKNOWN The update frequency is not known.
